Introduction

This page describes the ESP8266 Sensor that was built for use in a LoRa network/

Pin-out on HopeRF rfm95 shield:

  Top (antenna)  
DIO4   GND
DIO3   GND
DIO2   GND
DIO1   3.3V
DIO0   GND

MISO

   GND

MOSI

   RST
SCK   3.3V
NSS   3.3V
RST   3.3V

This solutions is not usable out of the box with the LoRaBee type of adapter board as that board does not expose the DIO5 pin of the RFM95.
